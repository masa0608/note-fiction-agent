# 感情の預かり人

## タイトル案

1. 感情の預かり人
2. 谷中、空白の六時間
3. 記憶の裏側で鳴るもの
4. 灰色の窓辺
5. あなたの涙を持っています

## 本文

　谷中の路地は、令和の空気を拒むように静かだった。

　五月の陽光が古い木造家屋の格子窓に斜めに差し込み、埃を金色に染めている。凛はオルゴールから見つけた紙片を握りしめたまま、スマートグラスに表示された地図と目の前の景色を照らし合わせた。

　住所は確かにここを指している。だが、そこにあったのは空き家だった。

「感情データ、検出不可。周辺三十メートル以内に登録済み市民なし」

　イヤーカフから流れるCARE-7の声が、凛の期待を静かに打ち消す。

　凛は錆びた門扉に手をかけた。触れた瞬間、指先に微かな振動を感じる。ポケットの中で、オルゴールが震えていた。

　昨夜と同じだ。この場所に反応している。

　門扉は驚くほど軽く開いた。油を差した形跡がある。誰かが最近、ここに出入りしている。

　玄関の引き戸には「藤堂」という古い表札がかかっていた。インターホンは見当たらない。凛は二度、戸を叩いた。

　返事はない。

　三度目に手を上げたとき、引き戸が内側から開いた。

「やっと来たね」

　白髪の老女が立っていた。八十歳は超えているだろう。深い皺の刻まれた顔に、不思議と穏やかな笑みを浮かべている。

「あの、私は——」

「霧島凛さん。知ってるよ。ずっと待ってた」

　凛の言葉を遮るように、老女は身体を横にずらして家の中を示した。

「上がんなさい。冷えるから」

　五月の晴れた日に、冷えるも何もない。だが凛は、自分の身体が震えていることに気づいた。恐怖ではない。この感覚は、何だ。

「CARE-7、バイタルチェック」

　小声で指示を出す。

「心拍数上昇。発汗量増加。感情データ……解析不能」

　解析不能。またこのエラーだ。昨夜、オルゴールが鳴ったときと同じ。

　凛は靴を脱ぎ、老女の後に続いた。

　家の中は時間が止まっているようだった。黒電話、ブラウン管テレビ、振り子時計。2028年の東京にあるはずのないものばかりが並んでいる。だが不思議と、埃っぽさはない。誰かが丁寧に暮らしている痕跡がある。

　老女は凛を六畳の和室に通した。卓袱台の上には、すでに二人分の湯呑みが用意されていた。

「待ってたって、どういうことですか」

　凛は卓袱台を挟んで座りながら問うた。

「その手紙を受け取ったら、あんたは必ずここに来る。そう聞いてたから」

「誰に」

「椿に」

　凛の心臓が、一度だけ大きく跳ねた。

　椿。妹の名前だ。

「……妹を、ご存じなんですか」

「知ってるも何も」老女は湯呑みを両手で包み込むようにして持った。「あの子は五年前、この家に来たんだよ。事故の日に」

　凛の視界が歪んだ。いや、歪んでいるのは現実のほうだ。

　事故当日、椿は一人で電車に乗っていたはずだった。帰省先から東京に戻る途中、脱線事故に巻き込まれた。凛は当時、学会出張でシンガポールにいた。妹の死を知ったのは、十二時間後のことだ。

「椿が、ここに？　でも、事故に——」

「事故には遭った。でもその前に、ここに来たんだ」

　老女は立ち上がり、押し入れの襖を開けた。中から古い木箱を取り出す。

「あの子が預けていったものがある」

　木箱の蓋が開く。

　中には、透明な球体が入っていた。直径十センチほど。水晶のように見えるが、内側に微かな光の粒子が漂っている。感情可視化技術の初期プロトタイプに似ていた。今は使われていない、旧式の——

「まさか」

「あんたが開発に関わった、最初期の感情保存デバイス。試作七号機。市場に出る前に破棄されたはずのもの」

　凛は自分の手が震えているのを見た。

　試作七号機。確かに存在した。感情をデータ化するだけでなく、一時的に「抽出」することが可能な、危険な技術。倫理委員会の勧告で開発が凍結され、すべてのデータと試作機が破棄されたはずだった。

　なぜ、それを椿が持っている。

「あんたの感情が入ってる」

　老女の声は淡々としていた。

「五年前の夏、あんたは妹にこれを預けた。『しばらく持っていてほしい』と」

「嘘です」

　凛は首を振った。

「私にはそんな記憶が——」

　言いかけて、止まった。

　六時間の空白。事故当日、どうしても思い出せない六時間。シンガポールのホテルで目覚める前の記憶が、ごっそり欠落している。

「記憶がないのは、感情と一緒に封印したからだよ」

　老女は球体を凛の前に置いた。

「椿は言ってた。『姉は、いつか自分で取りに来る。そのときまで絶対に渡さないでほしい』って」

　凛は球体を見つめた。内側の光の粒子が、かすかに脈打っている。まるで心臓のように。

「CARE-7」

　声が掠れた。

「この球体内のデータ、解析できる？」

「試行中……」数秒の沈黙の後、CARE-7が答えた。「感情データを確認。登録者：霧島凛。記録日時：2023年8月15日14時23分。感情種別：悲哀、愛着、恐怖、後悔——複合感情パターン。保存状態：完全」

　2023年8月15日。椿が死んだ日の、六時間前。

　凛は椿が死ぬ前に、自分の感情を預けていた。

「……なぜ」

　自分が何を言っているのか、わからなくなっていた。

「なぜ私は、そんなことを。椿が死ぬとわかっていたわけでもないのに」

「それは、あんたにしかわからない」

　老女は立ち上がり、窓辺に向かった。

「でも、椿は一つだけ言い残してたよ」

　振り子時計の音だけが、室内に響いている。

「『姉は、私が死ぬことを知っていた』って」

　凛の全身から力が抜けた。

　知っていた？　私が？　椿の死を？

「それは——」

「私に聞かれても困る」老女は窓の外を見たまま言った。「あんたの感情は、そこにある。取り戻すかどうかは、あんた次第だ」

　凛はゆっくりと球体に手を伸ばした。

　触れる直前で、止まった。

　ポケットの中で、オルゴールが再び鳴り始めていた。今度は昨夜と違うメロディだ。聞いたことがある。椿が好きだった曲——「亡き王女のためのパヴァーヌ」。

　同時に、CARE-7が警告を発した。

「緊急検出。感情データの外部流入を確認。発信源……特定不能。ただし、パターンは登録者『霧島椿』と98.7％一致」

　凛は弾かれたように顔を上げた。

　椿の感情データ？　五年前に死んだ妹の？

「まさか……」

　窓の外で、何かが動いた気がした。人影のような、光の残像のような。

　老女が静かに振り返った。

「あの子は、まだここにいるのかもしれないね」

　オルゴールの音が、止まった。

　代わりに、凛のスマートグラスに一件の通知が表示された。

　差出人不明。

　件名は、たった一行。

『お姉ちゃん、思い出して。あの日、私が何を頼んだか』

　凛の指先が、無意識に自分の胸を押さえた。

　何も感じないはずの心臓が、痛みのようなものを訴えている。

　灰色だったはずの心臓が、確かに今、鳴っていた。
