# 文体ルール

## 守ること

- 読みやすい日本語
- 情景が浮かぶ描写
- 1文を長くしすぎない（目安：60字以内）
- 心理描写を入れる
- 会話文を適度に使う
- 最後の一文に余韻を残す
- 「続きを読みたい」と思わせる章末

## 避けること

- 説明が多すぎる
- 設定を語りすぎる
- キャラクターが薄い
- AIっぽい一般的な比喩ばかり
- どこかで見たような展開だけで終わる
- オチのためだけの物語

## 文字数

- 平日：2,000〜3,500字
- 土日：4,000〜6,000字
